  1. Scan the QR above. Safari downloads a profile — tap Allow (and Continue if Safari warns the site isn't secure).
  2. Settings → General → VPN & Device Management → drivecast CA → Install.
  3. Before tapping Install, open More Details and check the SHA-256 fingerprint matches the one below — if it doesn't, someone on your network is tampering with the download; do not install it.
